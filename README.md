# 🚀 Mini CRM - Smart Lead Manager

A simple and elegant CRM web app to manage leads with a modern dashboard UI.

## ✨ Features
- ➕ Add new leads
- 🔍 Search leads
- 🏷️ Update lead status (New, Contacted, Closed)
- 📝 Add notes
- ❌ Delete leads
- 🌙 Dark / Light mode toggle
- 🎨 Clean and professional dashboard UI

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
